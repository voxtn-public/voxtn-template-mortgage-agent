# voxtn-template-mortgage-agent

Vox Agents template for **mortgage agents**.

**Primary use case:** KaranSekar.com (Karan Sekar, DLC Group) — Ontario mortgage
brokerage, DLC-aware application workflows, commission tracking, client follow-up.

**Capabilities (planned):**
- DLC-compliant client intake flow
- Lender-matching draft logic
- Commission + pipeline tracking
- Follow-up cadence automation

**Status:** v0 scaffold (Phase 1a 2026-05-20). Content (`SOUL.md`, `skills/`, `config.yaml`) added in Phase 1d.
